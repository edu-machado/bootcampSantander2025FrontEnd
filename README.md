# bootcampFrontEndSantander2025
Repositório com os estudos e projetos do Bootcamp Santander 2025 Front-End da DIO. Aprendizado focado em HTML, CSS (Flexbox e Grid) e JavaScript, com consumo de APIs, interatividade e responsividade. Inclui resumos de conteúdo ministrado, mentorias e desafios de código com 10 projetos práticos com temas como Stranger Things, Yu-Gi-Oh e mais.
